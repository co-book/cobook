## Development Guidelines

| Application Framework | Spring Framework |
|-----------------|----------------|
| Web Application Server | Tomcat 8.0 |
| Language | Java |
| Build Tool | Maven |

### Commit Rules
> feat: 기능 추가  
> fix: 오류수정  
> docs: 문서파일 수정  
> perf: 성능개선코드  
> refactor: 리팩토링  
> test: 누락된 테스트코드  
> chore: 잡다변경 ( 포멧팅, 설정변경 등) 
```
# 페이지 내 기능 추가
feat(modify): 게시물 수정 기능 추가
```
